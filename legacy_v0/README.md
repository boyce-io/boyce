# legacy_v0 — Quarantined v1.0.0 Code

This directory contains the original DataShark v1.0.0 Python package (`src/datashark/`), the DataShark_Agents sub-project, the DataShark_Client VS Code extension, the DBeaver plugin, associated tests, legacy deployment scripts, and ZIP release archives. This code is **no longer the active deliverable** and is preserved here for historical reference only.

**The primary deliverable is [`../datashark-protocol/`](../datashark-protocol/).** All new development happens there.
